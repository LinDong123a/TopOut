# TopOut 🧗

攀岩工具 App — 中国攀岩者的一站式成长记录平台

## 产品定位

**戴着手表去爬，其他全自动。**

## 应用矩阵

- ⌚ **watchOS App** — 攀爬时核心，自动检测攀爬/打卡/记录
- 📱 **iOS App** — 爬前爬后主力，数据分析/社交/内容
- 🧩 **Widgets** — 锁屏/桌面速览
- ☁️ **Python Backend** — 数据同步/AI分析引擎

## 核心功能

### 自动检测
- 传感器自动检测上墙/落地
- 自动记录攀爬次数/时间/心率/高度曲线
- GPS 地理围栏自动打卡岩馆

### 实时大屏
- iPhone 实时显示攀爬数据（心率/难度/统计）
- 搭子围观模式

### 成长追踪
- 攀爬日历热力图
- 难度曲线 + 里程碑
- 统计仪表盘

### 社交
- 搭子系统（小圈子强关系）
- 岩馆动态（谁在哪/约人）
- 攀爬视频分享 + Beta 讨论

### AI 功能（付费）
- 视频复盘（动作分析/关键帧标注）
- 个性化训练计划
- 弱项分析 + 雷达图

## 技术栈

- iOS/watchOS: Swift + SwiftUI + SwiftData + Core Motion + HealthKit
- Backend: Python + FastAPI
- AI: Core ML (端侧) + Cloud AI (服务端)
